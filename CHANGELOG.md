# Changelog

## [v1.2.0] - 2026-04-03
### 🛠 Critical Fixes
- **Python:** Исправлен синтаксис в блоке `__main__`, который приводил к краху при запуске.
- **Imports:** Добавлен отсутствующий `import psutil`, необходимый для мониторинга нагрузки CPU.
- **ESP32 Memory:** Замена `String` на `char[]` массивы в прошивке для предотвращения утечек памяти и перезагрузок.
- **Logic:** Улучшена обработка ошибок и проверка зависимостей перед запуском.

## [v1.1.0] - 2026-04-02
### ✨ Features
- **Daemon Mode:** Добавлен режим демона с логированием в файл и авто-рестартом.
- **Systemd:** Поддержка создания systemd-сервиса для автозапуска в Linux.
- **LCD2004:** Обновлена прошивка для Arduino Nano с поддержкой 4-строчного дисплея.
- **Sensors:** Поддержка `lm-sensors` и чтение температур из `/sys/class/hwmon`.

## [v1.0.0] - 2026-04-01
### 🚀 Initial Release
- Базовый мониторинг температуры CPU через USB.
- Поддержка ESP32 и Arduino Nano.
- Интерактивный выбор порта.
